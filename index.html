Projects

Ramen-Ingredient Analyzer

A tool to scan and analyze the ingredient lists on food packets, identifying potential non-vegetarian additives and other ingredients of interest.Built a full-stack web app that scans ramen package images using OCR (Tesseract) and classifies ingredients as vegetarian or non-vegetarian using Gemini Pro API. Used Python, Flask, and Pandas for backend processing and prompt-based classification; developed a custom HTML, CSS, and JavaScript frontend.

Mood Journal

Mood Journal is a full-stack MERN application that allows users to record, track, and analyze their moods over time. The app integrates sentiment analysis to automatically detect the emotional tone of each journal entry and visualizes mood patterns using dynamic charts. It focuses on creating a simple, responsive, and engaging way to reflect on mental well-being. It features CRUD operations for entries, integrated sentiment analysis for mood tracking, and visualized mood trends over time using React Chart.js
